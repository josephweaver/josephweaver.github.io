## How To Read These Notes

Concept notes are allowed to be incomplete. Read them as working hypotheses, not finished claims.

Useful questions while reading:

- What is the core object being represented?
- What uncertainty, provenance, or structure is being preserved?
- What would make the idea testable?
- Is this a research project, an essay, a prototype, or just a useful analogy?

When a concept matures, it can move into a focused project folder under `research/` or become a dated public post.
